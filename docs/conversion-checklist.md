# Conversions-Checkliste

Ziel dieser Checkliste ist **Vertrauen durch Ruhe und Klarheit** – ausdrücklich nicht durch Verkaufsdruck, künstliche Verknappung oder Dropshipping-typische Taktiken.

## Produktseite

* [ ] Mehrere hochwertige Produktbilder aus unterschiedlichen Blickwinkeln
* [ ] Produktbeschreibung folgt der [Vorlage](product-description-template.md) (80–150 Wörter, ruhiger Ton)
* [ ] Materialangaben vorhanden und korrekt (keine erfundenen Details)
* [ ] Klarer, unaufdringlicher Call-to-Action
* [ ] Keine künstliche Verknappung ("nur noch 2 auf Lager" nur, wenn faktisch zutreffend)
* [ ] Keine erfundenen oder unbelegten Kundenbewertungen

## Vertrauenselemente

* [ ] Sichtbarer Zugang zu Kontaktinformationen
* [ ] Sichtbarer Zugang zu Versand- und Rückgabeinformationen (sobald final, siehe [Rechtliche Platzhalter](legal-placeholders.md))
* [ ] Über-uns-Bereich mit echter Markengeschichte statt generischem Text
* [ ] Konsistente, ruhige Bildsprache über die gesamte Seite hinweg

## Sprache & Ton

* [ ] Kein aggressiver Verkaufston, keine Ausrufezeichen-Häufung
* [ ] Keine Floskeln wie „zeitlos elegant" oder „ein Must-have" (siehe [Markenidentität](brand-identity.md))
* [ ] Keine unbelegten Superlative ("Bestseller", "meistgeliebt") ohne Datenbasis

## Technisch

* [ ] Ladezeit der Produktseiten geprüft
* [ ] Mobile Darstellung geprüft
* [ ] Checkout-Prozess getestet (keine Reibung, klare Schritte)

{% hint style="info" %}
Diese Liste ist bewusst konservativ gehalten. Sie priorisiert langfristiges Vertrauen gegenüber kurzfristiger Konversionsoptimierung durch Druckmittel.
{% endhint %}
