# Nächste Schritte

## Wichtigster Punkt zuerst: eigener Shop

Der aktuell verbundene Shopify-Shop (**„AURÈLE"**, `aurele-2045.myshopify.com`) ist ein geteilter Multi-Vertical-Testshop — neben Jewelkunst-/AURÈLE-Schmuck laufen dort parallel NOCTRA (Schlafmasken), PULS (Massagegerät), CRUMBL (Küchenmaschine), TRAILPAW (Hundezubehör), LUMENA (Beauty-Gerät) und NOEMBRA (Wohnaccessoires). Für eine seriöse Premium-Schmuckmarke sollte Jewelkunst einen **eigenen, dedizierten Shop** bekommen.

**Exakte Schritte:**

1. Neuen Shopify-Shop anlegen (eigenes Konto/eigene `myshopify.com`-Subdomain), Plan wählen
2. Domain `jewelkunst.net` mit dem neuen Shop verbinden (Shopify Admin → Settings → Domains)
3. Die 15 aktiven Jewelkunst-/AURÈLE-Schmuckprodukte aus dem AURÈLE-Shop in den neuen Shop übertragen (Export/Import über Shopify's Produkt-CSV-Export, oder manuell neu anlegen — Produktdaten liegen bereits vollständig vor)
4. Kollektionen neu anlegen: Halsketten, Armbänder, Ohrringe, Ringe, Bestseller, Geschenkideen, Fußkettchen (Tag-Regeln siehe bestehende Collections im AURÈLE-Shop als Vorlage)
5. Theme wählen, das zur ruhigen, minimalistischen Markensprache passt (kein Dropshipping-Theme mit Countdown-Timern/Pop-ups)
6. Inhalte aus diesem GitBook-Space (`gitbook/`) einpflegen: [Startseite](homepage.md), [Kategorie-Seiten](category-pages.md), Produktseiten nach [Vorlage](product-page-template.md)
7. Rechtliche Seiten erst veröffentlichen, wenn alle Platzhalter in [legal-placeholders.md](legal-placeholders.md) durch geprüfte Angaben ersetzt sind

{% hint style="danger" %}
Zwei unterschiedliche Vendor-Namen im aktuellen Katalog — „AURÈLE" (dark-luxury-positionierte Produkte wie LUNARIA, ÉTOILE, VELA) und „Jewelkunst" (schlichtere, deutsch benannte Produkte wie Damen Ankerkette, Ring Bonafide) — sollten vor dem Umzug in den neuen Shop unter **einer** Markenlinie vereinheitlicht werden. Aktuell ist unklar, ob beide Linien zu Jewelkunst gehören sollen oder ob AURÈLE eine separate Marke bleibt. Das ist eine Entscheidung, die vor Schritt 3 getroffen werden sollte.
{% endhint %}

## Priorität: Was muss heute gemacht werden, um Umsatz zu ermöglichen?

Vollständige, sortierte Prioritätenliste siehe [Conversions-Checkliste](conversion-checklist.md). Kurzfassung der wichtigsten Sofort-Schritte:

1. **Entscheidung eigener Shop** treffen (siehe oben) — alles Weitere hängt davon ab
2. **Rechtliche Platzhalter ausfüllen** ([legal-placeholders.md](legal-placeholders.md)) — ohne gültiges Impressum/Widerrufsrecht kein rechtlich sauberer Launch in Deutschland
3. **Startseite** einpflegen ([homepage.md](homepage.md))
4. **Produktseiten vervollständigen** für die 15 aktiven Schmuckprodukte ([product-page-template.md](product-page-template.md))
5. **Social-Media-Profile einrichten** mit Bio-Texten aus [social-media-plan.md](social-media-plan.md)
