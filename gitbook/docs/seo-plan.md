# SEO-Plan

{% hint style="warning" %}
Dieser Plan enthält bewusst **keine** recherchierten Keyword-Daten, Suchvolumen oder Rankings – diese liegen aktuell nicht vor. Er beschreibt stattdessen die Leitplanken und Struktur, nach denen eine spätere Keyword-Recherche eingeordnet werden sollte.
{% endhint %}

## Grundprinzipien

* Deutsche Sprache durchgängig, konsistent mit der Markentonalität (siehe [Markenidentität](brand-identity.md))
* Keywords dienen der Auffindbarkeit, dürfen aber die ruhige, elegante Tonalität nie verdrängen – kein Keyword-Stuffing
* Material- und Edelstein-Begriffe (z. B. konkrete Steinnamen) sind natürliche Ankerpunkte, sofern sie tatsächlich zutreffen

## Struktur je Seitentyp

### Produktseiten

* Title Tag: [Platzhalter-Formel, z. B. „{Produktname} – {Material} | Jewelkunst"]
* Meta Description: [Platzhalter – 150–160 Zeichen, im Markenton, ohne erfundene Verkaufsversprechen]
* Bild-Alt-Text: beschreibend und faktisch (Material, Stücktyp), keine Keyword-Häufung

### Kollektionsseiten

* Title Tag: [Platzhalter]
* Einleitender Text: kurz, im Markenton, verlinkt auf einzelne Produkte

### Über-uns-Seite

* Fokus auf Markengeschichte und Handwerk statt auf Keyword-Dichte

## Offene Punkte

* [ ] Keyword-Recherche durchführen (Tool: [Platzhalter])
* [ ] Wettbewerbsanalyse: [Platzhalter]
* [ ] Technisches SEO-Audit (Ladezeit, strukturierte Daten) – separat zu planen
* [ ] Interne Verlinkungsstruktur zwischen Kollektionen und Produkten festlegen

{% hint style="info" %}
Sobald echte Keyword-Daten vorliegen, sollten sie als eigener Abschnitt ergänzt werden, statt Platzhalter durch geschätzte Zahlen zu ersetzen.
{% endhint %}
