# Lead Generation

**Datenquellen (in Prioritätsreihenfolge)**
1. **Google Maps / Google My Business** — Hauptquelle. Liefert Name, Adresse, Telefon, Website, Bewertungen, Fotos, Öffnungszeiten.
2. **Das Örtliche / 11880 / Gelbe Seiten** — Backup für Firmen ohne starke GMB-Präsenz, liefert oft direkte E-Mail-Adressen.
3. **IHK/HWK Unternehmensregister Stuttgart & Karlsruhe** — für verifizierte Handwerksbetriebe (Innung-Mitgliedschaft = finanziell stabiler).
4. **LinkedIn (Sales Navigator oder normale Suche)** — für Inhaber/Geschäftsführer-Kontakte + Firmengröße.
5. **Instagram Hashtag/Standort-Suche** — für Solar/Handwerksbetriebe, die bereits Social-Media aktiv nutzen (= Outreach-Kanal bereits vorhanden).
6. **Facebook Ad Library** (facebook.com/ads/library) — check ob Konkurrenz/Agentur bereits für sie schaltet.

**Konkrete Suchbegriffe (Google Maps)**
Kombiniere Branche × Stadt. Branchenliste:
- Dachdecker
- Photovoltaik Installateur / Solarteur / PV-Anlagen Fachbetrieb
- Elektriker / Elektroinstallateur
- Heizungsbau / SHK Betrieb
- Sanitärinstallateur
- Maler- und Lackierbetrieb
- Fliesenleger
- Schreinerei / Tischlerei
- Garten- und Landschaftsbau
- Fensterbau / Fensterbetrieb

Städte-Liste: Stuttgart, Karlsruhe, Böblingen, Esslingen, Ludwigsburg, Pforzheim, Heilbronn, Reutlingen, Tübingen, Waiblingen, Sindelfingen, Bruchsal, Ettlingen.

Beispiel-Suchstrings für Google Maps:
```
Photovoltaik Installateur Stuttgart
Solarteur Karlsruhe
Dachdecker Böblingen
Heizungsbau Esslingen
Elektriker Ludwigsburg
```
→ Trage jede Kombination einzeln in Google Maps ein (nicht "Handwerker Stuttgart" — zu unspezifisch, schlechte Ergebnisse).

**Filterlogik — nur Leads mit allen folgenden Kriterien aufnehmen:**
- Google-Bewertung zwischen 4.0 und 4.8 Sternen (5.0 = meist zu wenige Reviews, unter 4.0 = Reputationsproblem, das dein Angebot nicht löst)
- Mindestens 15 Bewertungen (= etablierter Betrieb, kein Start-up ohne Budget)
- Aktivste Bewertung nicht älter als 3–4 Monate (= Betrieb ist aktiv, hat Kundenverkehr)
- Website existiert, aber: kein klares Lead-Formular, keine Terminbuchung, veraltetes Design, oder keine Google Ads laufen (Ad Library check)
- Firmengröße geschätzt 5–30 Mitarbeiter (via LinkedIn/Website-Team-Seite — zu klein = kein Budget, zu groß = eigene Marketingabteilung)
- Telefonnummer + möglichst Ansprechpartner-Name auffindbar
- Ausschluss: Franchise-Betriebe, Konzerntöchter, Betriebe mit bereits sichtbar professionell laufenden Ads (Facebook Ad Library zeigt aktive Kampagne einer Agentur)

**Tool-Tipp:** Trage Ergebnisse direkt ins CRM-Sheet ein (Struktur siehe [CRM](crm.md)), nicht in eine separate Liste — spart einen Schritt.
